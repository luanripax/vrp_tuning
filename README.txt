Instruções para uso de vrp_tuning :

Comando principal: /tunar

Teclas e funções: Backspace: retorna para seção anterior
 		  Esc: fecha a interface

- Para habilitar o som, vá em "config.lua" e mude a propriedade de "Config.sound" para true.

- Para adicionar categorias e peças, vá em "config.lua" e adicione nas respectivas listas.

- O script não suporta tunagem de peças com sub-categorias como cores,rodas, etc.

- Para reproduzir os sons é necessário inseri-los em vrp_sounds > html > sounds .

- Para que as imagens apareçam, altere o ip contido no arquivo "ui.js" .
